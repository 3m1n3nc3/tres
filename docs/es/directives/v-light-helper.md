# v-light-helper 🔆

Con la nueva directiva v-light-helper proporcionada por **TresJS**, puedes agregar rápidamente el ayudante respectivo a tus luces con solo una línea de código 😍.

Se admiten las siguientes luces:
- DirectionalLight
- PointLight
- SpotLight
- HemisphereLight
